- Linux systems

# Phase 1 — Linux Systems & Bash

**Goal**: Build strong comfort with Linux tooling and shell scripting.

## Lasso
> A command line tool that gathera & shows basic network info


**Skills learned**:

- Bash scripting
- Linux networking commands
- CLI design
- parsing command output
- modular shell scripts

**Example Commands:**

- lasso ip
- lasso dns
- lasso routes
- lasso interfaces
- lasso summary

 **Tools used:**
```
ip
hostnamectl
resolvectl
awk
grep
```
